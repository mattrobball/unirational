# Problem C — Counterexample sweep for the Debarre–de Jong conjecture at d = 9

> **Audit note (2026-07-22).** This is the original search brief, not the
> final certificate. Several field, citation, and verification claims below
> require correction. See [`RESOLUTION.md`](RESOLUTION.md): the literal
> field-free statement is refuted in characteristic two, while the intended
> characteristic-zero problem remains open and is reduced there to a sharply
> constrained sixfold problem.

## Convention

Literature notation: X ⊂ P^N a SMOOTH hypersurface of degree d. The Fano
scheme of lines F_1(X) ⊂ G(1, N) has expected dimension 2N − d − 3
(dim G(1,N) = 2(N−1), minus d+1 conditions). Harris's notes state this as
the "Question (proposed by de Jong)" on p. 27 in the shifted notation
X ⊂ P^{n+1}, expected dim = 2n − 1 − d; identical content (N = n+1).

## Mathematical statement

**Conjecture (Debarre–de Jong).** For every smooth X_d ⊂ P^N with d ≤ N,
dim F_1(X) = 2N − d − 3.

Unlike Problems A/B this quantifies over ALL smooth hypersurfaces, not the
general one — which is exactly what makes it a pure counterexample hunt:

- A single smooth X with dim F_1 ≥ 2N − d − 2, certified over Q, refutes it.
- No amount of sampling can confirm it; the positive direction is theory
  only. This sweep can therefore only ever end in a refutation (major
  theorem) or a documented null result (evidence + reusable infrastructure).

**Status (verified July 2026).** Proven: d ≤ 3 classical; d = 4 Collino;
d = 5 Debarre; d ≤ 6 and then d ≤ 8 Beheshti; N ≥ 2d − 4 Beheshti–Riedl.
**Open frontier: exactly d = 9 with N ∈ {9, 10, 11, 12, 13}.**
Sharpness of the conjectured bound: cones always have excess F_1 but are
singular; the smooth Fermat of degree N+1 in P^N has dim F_1 = N − 3 >
2N − (N+1) − 3, i.e. excess appears immediately past d = N. The conjecture
says smoothness prevents excess up to d = N on the nose.

## Per-candidate verification (fully algorithmic)

Given F ∈ F_p[x_0..x_N]_9 (or over Q):

1. **Smoothness**: the Jacobian ideal J = (∂F/∂x_0, …, ∂F/∂x_N) has empty
   projective vanishing locus. Screen: msolve on the affine cone slices;
   certify: M2 `dim (ideal jacobian F) <= 0` (i.e. only the irrelevant
   ideal). Over F_p this certifies smoothness of the reduction only.
2. **dim F_1**: on the standard chart of G(1, N), lines are
   span(u, v) with u = (1, 0, a_2..a_N), v = (0, 1, b_2..b_N);
   F(s·u + t·v) ≡ 0 gives d+1 = 10 equations in the 2(N−1) chart variables.
   Lines avoiding the chart form a measure-zero set handled by re-running in
   2–3 random PGL-translated charts (or a Plücker-based run for final
   certification).
   Screen for "dim ≥ k": intersect with k random hyperplanes in chart
   coordinates and test nonemptiness (0-dimensional msolve run). At the
   expected dimension the (expected+1)-slice test comes back empty; a
   nonempty result flags a candidate. Certify dimensions with an honest
   Grobner `dim` run.

**The characteristic caveat (README certificate 3).** Excess dimension is a
closed condition: an F_p excess example may not lift. Protocol for any hit:
re-verify at a second large prime → attempt small-height Q-lift and run the
dimension computation in exact arithmetic → only a Q-certified smooth excess
example is a counterexample. Small/wild primes (p = 3 is wild for d = 9)
are allowed as *shape generators* but never as evidence.

## Search design

Brute force is out (C(18,9) = 48620 monomials in the N = 9 coefficient
space), and unstructured dense-random X will have expected dimension with
overwhelming probability (that is a theorem for general X — classical for
lines). The sweep lives or dies on structured ansatze. Guardrails first:

**Provable dead ends (record as lemmas in the log, don't search them).**
- Linear spaces: a smooth hypersurface of dimension n = N−1 contains linear
  P^k only for k ≤ n/2. At N = 9: P^4 allowed, and lines inside a P^4 give
  dim G(1,4) = 6 = expected exactly — the maximal linear space *saturates*
  the conjecture without breaking it; P^5 (which would give
  dim G(1,5) = 8 > 6) is impossible.
- Line-rich subvarieties cut by few equations force singularities: if
  W ⊂ X with I_W generated by c elements with independent differentials
  along W, then F = Σ A_j G_j is singular along {A_1 = … = A_c = 0} ∩ W,
  nonempty whenever c ≤ dim W. Example: X ⊇ smooth 5-dim quadric
  (lines: dim 7 > 6) needs c = 4 ≤ 5 ⟹ X singular. This is the structural
  reason the conjecture resists: excess-line configurations want to sit on
  low-codimension-cut subvarieties, and those collide with smoothness.
  Corollary for search design: viable ansatze must place excess lines on
  loci of HIGH ideal codimension (many generators, big Rao modules) or
  spread them without a containing subvariety at all.

**Ansatz families (in priority order).**
1. **Invariant hypersurfaces**: degree-9 invariants of finite subgroups
   G ⊂ PGL_{10} (permutation + monomial groups, products of small unitary
   reflection groups, Heisenberg-type groups where order permits).
   Extremal Fano schemes historically concentrate on symmetric objects
   (Fermat!). Enumerate low-dimensional invariant families, sweep them
   entirely — a family of dim ≤ 5–6 can be swept exhaustively over small
   F_p by slicing.
2. **Sparse/binomial-rich forms**: Fermat plus few terms,
   Σ x_i^8 x_{σ(i)} (Klein-like cycles), sums of 9th powers of linear forms
   in few variables + perturbation. Cheap to sample in bulk; smoothness
   fails often, so filter smoothness first (it's the cheaper check for
   sparse F).
3. **Smoothed near-cones**: F = G(x_0..x_8) + Σ_{i} x_9 H_i-type
   deformations of cones over degree-9 hypersurfaces in P^8 (cones have
   excess; generic smoothing kills it; hunt the non-generic smoothings by
   solving for deformations preserving a chosen line family to first
   order — a linear-algebra condition worth implementing exactly).
4. **Mod-p exotica as shape generators**: sweep p ∈ {2, 5, 7, 11, 13} sparse
   forms for excess; any pattern found guides Q-ansatze (never evidence by
   itself).

## Cost model

Per candidate at N = 9, p = 32003: smoothness screen (msolve, 10 degree-8
equations in 10 vars) — seconds to minutes for sparse F; slice test for
dim F_1 ≥ 7 (10 degree-9 chart equations + 7 linear, 16 vars) — expect
minutes; full `dim` certification only for flagged candidates. Thousands of
structured candidates are feasible; log every (F, p, seed, chart, outcome)
row. The N ≥ 10 columns roughly double per step; run them only on ansatze
that look interesting at N = 9.

## Outcomes

- **Null (likely)**: a documented, certified sweep log over the structured
  families — genuine evidence for the open frontier of a 25-year-old
  conjecture, and the invariant-family Fano-scheme dimensions are of
  independent interest (extremal but not excessive examples, cf. the
  Beheshti–Starr "too many lines" refinement in arXiv:0810.4158).
- **Hit**: after the Q-certification protocol, a refutation of Debarre–de
  Jong — immediately check d = 9 minimality against the literature and
  compute the excess component's geometry (is it swept by a recognizable
  subvariety? that structure is the theorem).
